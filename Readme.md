# 🏅 Olympic Games Medal Analysis (Tableau)

📊 This project analyzes **Olympic Games medal data** using **Tableau** to discover patterns in medal distribution across countries, sports, and Olympic years.

The dashboard helps visualize which countries dominate the Olympics and how medal trends change over time.

---

## 📌 Project Overview

This Tableau dashboard provides insights into Olympic performance by analyzing:

🌍 Medal distribution across countries
🥇 Gold, Silver, and Bronze medal counts
🏃 Sport-wise medal contribution
📅 Year-wise Olympic trends

The goal of this project is to demonstrate **data visualization and exploratory data analysis using Tableau**.

---

## 📂 Dataset

The dataset used is the **Olympic Athlete Events dataset** which includes information about athletes and Olympic events.

### Important Fields Used

* 👤 Athlete Name
* 🌍 Country (NOC / Region)
* 🏃 Sport
* 📅 Year
* ⚧ Gender
* 🥇 Medal Type (Gold, Silver, Bronze)

---

## 📊 Dashboard Features

### 🌍 World Map Visualization

Shows medal distribution across different countries.

### 🏅 Country-wise Medal Count

Displays top-performing countries in Olympic history.

### 🏃 Sport-wise Medal Distribution

Identifies sports that produce the most medals.

### 📈 Year-wise Medal Trends

Tracks how Olympic performance changes over time.

## 🔍 Key Insights

* 🇺🇸 Some countries dominate Olympic medal counts historically.
* 🏃 A few sports contribute a large portion of total medals.
